Podsumowanie poznanych technik CSS (Kompletne)

Poniżej znajduje się lista wszystkich właściwości użytych w zadaniach 1-5. Elementy dodane teraz oznaczono jako (DODANE).

1. Podstawy i Selektory (Zadanie 1)

<style> / <link> / style="..."
Trzy metody dodawania CSS: wewnętrzna, zewnętrzna (zalecana) i liniowa (inline).
.klasa
Selektor klasy (kropka); wybiera wiele elementów.
#id
Selektor identyfikatora (płotek); wybiera jeden unikalny element.
* { ... } (DODANE)
Selektor uniwersalny. Używaliśmy go do resetowania Box Modelu (box-sizing: border-box).

2. Typografia i Wygląd Tekstu (Zadanie 2)

font-family
Krój czcionki (w tym import Google Fonts).
font-weight
Grubość znaków (np. 300, 400, 700).
text-align
Wyrównanie tekstu (left, center, right, justify).
line-height
Interlinia. Kluczowa dla czytelności długich bloków tekstu.
text-transform (DODANE)
Zmiana wielkości liter (uppercase - wielkie, capitalize - jak w nazwach własnych).
text-decoration (DODANE)
Dekoracja tekstu. Używane głównie do usuwania podkreślenia z linków (none) lub dodawania go na hover.
color: hsla(...)
Kolor w formacie Hue, Saturation, Lightness, Alpha (przezroczystość).

3. Box Model i Obramowania (Zadanie 3)

box-sizing: border-box
Zmienia sposób liczenia szerokości elementu, wliczając padding i border do środka.
padding / margin
Odstępy wewnętrzne (w środku ramki) i zewnętrzne (poza ramką).
border-radius (DODANE)
Zaokrąglenie rogów elementu (używane w kartach, przyciskach i inputach).
box-shadow (DODANE)
Cień rzucany przez element. Nadaje głębię (efekt uniesienia karty lub focusu w formularzu).
list-style-type (DODANE)
Zmiana punktorów listy (np. na kwadraty, rzymskie cyfry lub usunięcie ich całkowicie).

4. Layout, Pozycjonowanie i Efekty (Zadanie 4)

position: relative / absolute
Duet do precyzyjnego układania elementów (np. etykieta w rogu zdjęcia).
position: fixed / sticky
Elementy przyklejone do okna lub krawędzi podczas przewijania.
z-index
Zarządzanie warstwami (co przykrywa co).
display: flex (DODANE)
Użyliśmy tego w Lightboxie do idealnego wycentrowania obrazka na środku ekranu.
background: linear-gradient(...) (DODANE)
Tło jako płynne przejście między kolorami (użyte w tle strony i overlayu).
background-attachment: fixed (DODANE)
Efekt Parallax – tło "stoi" w miejscu podczas przewijania treści.
transition (DODANE)
Płynna animacja zmiany stanu (np. powolna zmiana koloru przycisku).
transform (DODANE)
Przekształcenia geometryczne (np. scale do powiększenia zdjęcia, translateY do wyśrodkowania).
:target (DODANE)
Pseudo-klasa reagująca na ID w adresie URL (serce naszego Lightboxa bez JavaScript).

5. Formularze i Zaawansowane Interakcje (Zadanie 5)

:hover / :focus / :active
Stany elementu: najechany, aktywny (pisanie), kliknięty.
:not(:placeholder-shown)
Sprawdza, czy pole nie jest puste (czy użytkownik coś wpisał).
:checked (DODANE)
Wykrywa zaznaczony checkbox/radio. Kluczowe do tworzenia własnego wyglądu przycisków wyboru.
:disabled (DODANE)
Stylizacja wyłączonego pola formularza.
:valid / :invalid (DODANE)
Walidacja poprawności danych (np. zielona/czerwona ramka).
::before / ::after
Pseudo-elementy do wstawiania dekoracji (gwiazdki, ikony, customowe checkboxy).
cursor: pointer / not-allowed
Zmiana wyglądu kursora myszy.
