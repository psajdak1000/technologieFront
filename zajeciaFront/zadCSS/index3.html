Box Model i Listy w Praktyce

1. Zalety języka HTML (Unordered)

  • Prosty do nauczenia się i użycia
  • Szeroko wspierany przez wszystkie przeglądarki
  • Umożliwia tworzenie struktury stron internetowych
  • Możliwość łatwej integracji z CSS i JavaScript
  • Standardowy język dla internetu

2. Kroki tworzenia strony (Ordered)

  1. Planowanie treści i struktury strony
  2. Projektowanie wyglądu strony
  3. Tworzenie kodu HTML i CSS
  4. Testowanie strony w różnych przeglądarkach
  5. Publikacja strony w internecie

3. Słownik pojęć (Definition List)

DOCTYPE
Instrukcja określająca wersję i typ dokumentu HTML
Tag
Pojedynczy element kodu HTML otoczony nawiasami ostrymi.
Atrybut
Informacja dodatkowa w tagu, która modyfikuje jego działanie.
Element
Całość znacznika HTML wraz z zawartością i atrybutami.

5. Demonstracja Box Model

Oba pudełka mają w CSS: width: 300px, padding: 20px, border: 10px.

content-box

Szerokość to tylko treść.

Rzeczywista szerokość:
300 + 40 (pad) + 20 (border) = 360px

border-box

Szerokość zawiera wszystko.

Rzeczywista szerokość:
Jest równa dokładnie 300px (treść się kurczy).
