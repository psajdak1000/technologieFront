3. Definicje terminów HTML

DOCTYPE
Instrukcja określająca wersję i typ dokumentu HTML
Tag
Pojedynczy element kodu HTML otoczony nawiasami ostrymi, np. <html>
Atrybut
Informacja dodatkowa w tagu, która modyfikuje jego działanie, np. href w <a>
Element
Całość znacznika HTML wraz z zawartością i atrybutami
Paragraf
Element tekstowy oznaczający akapit tekstu, reprezentowany przez tag <p>
